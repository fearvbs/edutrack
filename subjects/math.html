Математика

Учебные материалы по математике: формулы, задачи, примеры.

Примеры формул

  • Площадь круга: S = π·r²
  • Площадь прямоугольника: S = a·b
  • Формула квадрата суммы: (a+b)² = a² + 2ab + b²

Пример задачи

Найти площадь круга с радиусом 5 см. Решение: S = π·5² = 78.5 см².
