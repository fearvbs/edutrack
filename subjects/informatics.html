Информатика

Информатика изучает алгоритмы, программирование и обработку данных. На этой странице собраны примеры кода, алгоритмы и объяснения, которые пригодятся для выполнения школьных заданий.

Здесь вы найдете пошаговые инструкции и примеры на Python, которые помогут понять основные концепции программирования.

Пример алгоритма

Алгоритм нахождения суммы чисел от 1 до n:

sum = 0
for i from 1 to n:
    sum += i

Пример кода на Python

n = 10
sum = 0
for i in range(1, n+1):
    sum += i
print(sum)

Пример функции на Python

def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)

print(factorial(5))
